# Validation Report

Generated by running `scripts/run_validation_suite.sh`. This is portfolio
evidence of correctness, not merely an internal development gate — see
project spec Section 6, Phase 4.

## 1. CI Coverage (raw_ttest_ci)

**Target:** 93%–97% (nominal 95%), per spec Section 7.6.

**Configuration:** n_users=2000 per simulated experiment, baseline_rate=0.10,
true_effect=0.02, 1000 independent simulated experiments (base_seed=10000).

**Result:** observed coverage = **94.4%** (944/1000)

**Interpretation:** deviation from nominal 95% is ~0.8 SE (binomial SE at
this N ≈ 0.73pp) — consistent with sampling noise in the coverage estimate
itself, not evidence of miscalibration in Welch's t-test CI at this
configuration.

## 2. CUPED Variance Reduction Scaling

**Target:** variance reduction should scale with covariate_correlation
strength (spec Section 7.6): tested at correlation=0.3 vs 0.7, expecting
materially more reduction at 0.7.

**Configuration:** n_users=5000 per simulated experiment, baseline_rate=0.10,
true_effect=0.0 (isolates the covariate relationship from any effect
confound), 200 independent simulated experiments per correlation value.

**Result:**
- correlation=0.3: mean variance reduction = **9.07%** (theoretical: 9.0%)
- correlation=0.7: mean variance reduction = **48.95%** (theoretical: 49.0%)

Both measurements matched the theoretical `100×ρ²` prediction well within
one standard error (deviation of 0.95 SE at correlation=0.3, SE=0.074pp;
0.33 SE at correlation=0.7, SE=0.150pp) — confirming CUPED's variance
reduction is not just directionally correct but quantitatively tight at
this scale.

## 3. Known Limitations of This Validation

- CI coverage was tested at ONE configuration (n=2000, baseline=0.10,
  effect=0.02). Coverage at substantially smaller n or more extreme
  baseline rates is NOT validated here and may behave differently, since
  Welch's df is an asymptotic approximation.
- CUPED variance reduction was tested on a BINARY outcome
  (point-biserial correlation) with true_effect=0 — variance-reduction
  behavior under a nonzero true effect, or on a continuous metric, is not
  separately validated.
